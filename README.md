# CraveKart
CraveKart is a modern online food ordering platform designed to provide users with a seamless food delivery experience. The application allows customers to browse restaurants, explore menus, place orders, and track deliveries in real time through responsive interface. Built using React and REST API integration delivering a scalable user experience.
